# Game Plan — 2D 횡스크롤 플랫폼 액션 로그라이크

## Game Concept

2D 횡스크롤 시점에서 플레이어는 정교한 이동·점프로 스테이지를 돌파하며, 근접 콤보와 쿨다운 기반 스킬로 몰려오는 적을 처치한다. 스테이지를 클리어할 때마다 능력·아이템 보상을 선택해 매 판 다른 빌드를 쌓아가고, 죽으면 처음부터 다시 시작하는 로그라이크 구조다. 빠르고 경쾌한 액션의 손맛과, 보상 선택이 만들어내는 판마다 달라지는 전개가 핵심 재미다. Skul: The Hero Slayer의 경쾌한 근접 액션을 기본으로, Dead Cells의 속도감 있는 스테이지 진행과 Astral Ascent의 스킬 슬롯 구성을 차용한다.

## Reference Games

- **Skul: The Hero Slayer** (기본) — 참고 요소:
  - 2D 횡스크롤 플랫폼 이동·점프의 경쾌한 손맛
  - 근접 콤보 중심 전투
  - 스킬 2종 슬롯 + 쿨다운 운용
  - 스테이지 클리어 후 능력/아이템 보상 선택
- **Dead Cells** (차용) — 참고 요소:
  - 속도감 있는 스테이지 진행과 적 전멸형 방 클리어 감각
  - 죽으면 처음부터 시작하는 로그라이크 실행 흐름
- **Astral Ascent** (차용) — 참고 요소:
  - 스킬 슬롯 기반 능력 구성과 쿨다운 UI 표현
  - 스테이지 사이 보상 선택으로 빌드를 다듬는 방식

## Core Loop

```
[게임 시작 / 스테이지 진입]
   ↓
① 이동·점프 — 플랫폼을 이동하고 점프로 지형·적을 돌파 (땅 감지)
   ↓
② 전투 — 근접 콤보(2~3타)와 스킬(쿨다운)로 적을 공격
   ↓
③ 적 처치 — 추격+근접 적을 쓰러뜨림 (피격 시 플레이어 체력 감소)
   ↓
④ 스테이지 클리어 판정 — 적 전멸 시 클리어
   ↓
⑤ 보상 선택 — 2~3택 중 하나를 골라 다음 판 강화 → 다음 스테이지로
   ↓
(체력 0) → [사망 / 처음부터 재시작]
```

## Key Systems

각 시스템이 이후 구현의 클립 단위가 된다.

| 시스템 | 역할 |
|---|---|
| 플레이어 이동·점프 (2D Rigidbody) | 입력을 받아 2D 물리로 좌우 이동·점프시키고, 땅 감지로 점프 가능 여부를 판정한다 |
| 근접 콤보 공격 | 공격 입력을 받아 2~3타 콤보를 순서대로 발동하고 판정 범위 내 적에게 피해를 준다 |
| 스킬 (쿨다운) | 스킬 입력 시 능력을 발동하고 쿨다운을 관리한다 (MVP 1종) |
| 적 AI (추격+근접) | 플레이어를 추격하고 근접 범위에서 공격한다 |
| 체력 & 피격 & 사망 | 플레이어/적의 체력 관리, 피격 처리, 사망 판정 |
| 스테이지 클리어 판정 | 스테이지 내 적이 전멸했는지 판정해 클리어를 트리거한다 |
| 보상 선택 | 클리어 시 2~3개 보상 후보를 제시하고 선택을 적용한다 |

## Feature Priority

이 문서에서 가장 중요한 섹션이다.

### MVP — "한 스테이지가 성립하는 최소 로그라이크"

1. 2D Rigidbody 이동 + 점프 (땅 감지 포함)
2. 근접 콤보 2~3타
3. 스킬 1종 (쿨다운)
4. 적 AI 1종 (추격 + 근접) + 체력/피격/사망
5. 스테이지 1개 클리어 판정 (적 전멸)
6. 보상 선택 1회 (2~3택)

### 나중 (Later)

- 스킬 2종째 (두 번째 슬롯)
- 콤보 확장 (추가 타수·차지·에어 콤보)
- 함정 다양화 (가시·낙하 등 지형 위협)
- 절차적 스테이지 생성
- 해금 메타 성장 (영구 업그레이드)
- 다양한 적 종류 및 보스전
- 사운드·이펙트 폴리싱

## Development Order

의존 관계상 먼저 있어야 하는 것을 앞에 둔다.

1. **2D 이동 + 점프 (땅 감지)** — 나머지 모든 것의 기반. 캐릭터를 움직일 수 있어야 한다
2. **체력 & 피격 & 사망** — 플레이어/적이 죽을 수 있게. 전투·적 AI의 전제
3. **근접 콤보 공격** — 적을 처치 가능하게 (피격 시스템 위에 얹힘)
4. **적 AI (추격 + 근접)** — 상대할 대상 확보, 피격을 받아 죽는 적
5. **스킬 1종 (쿨다운)** — 전투 수단 확장
6. **스테이지 클리어 판정 (적 전멸)** — 한 판의 끝을 정의
7. **보상 선택 (2~3택)** — 클리어에 성장 보상 부여, 로그라이크 루프 완성
