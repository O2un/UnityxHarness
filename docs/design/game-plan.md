# Game Plan — 3D 뱀파이어 서바이버류

## Game Concept

탑다운 3D 시점에서 플레이어는 사방에서 몰려오는 적 무리를 자동 공격으로 처치한다. 직접 조작은 **이동뿐**이며, 처치로 얻은 경험치로 레벨업할 때마다 능력을 선택해 빌드를 쌓아간다. 시간이 갈수록 강해지는 적을 상대로 얼마나 오래 생존하며 강해지느냐가 핵심 재미다. Soulstone Survivors식 3D 탑다운 구성과 스킬 성장·저주 기반 난이도 조절을 지향한다.

## Reference Games

- **Soulstone Survivors** — 참고 요소:
  - 3D 탑다운 카메라 구성
  - 레벨업 시 스킬/패시브 선택 방식
  - 웨이브가 지날수록 강해지는 적 스케일링
  - (후기 확장용) 저주로 난이도↑·보상↑ 트레이드오프

## Core Loop

```
[게임 시작]
   ↓
① 이동 — 플레이어를 조작해 적을 피하거나 유리한 위치로
   ↓
② 적 접근 — 스폰된 적이 플레이어를 향해 몰려옴 (웨이브 진행)
   ↓
③ 자동 공격 — 사거리 내 적을 자동으로 공격해 처치
   ↓
④ 경험치 획득 — 처치한 적이 드롭한 경험치 수집
   ↓
⑤ 레벨업 시 능력 선택 — 일정 경험치 도달 시 강화 선택 → ①로 복귀
   ↓
(체력 0) → [사망 / 결과 화면]
```

## Key Systems

각 시스템이 이후 구현의 클립 단위가 된다.

| 시스템 | 역할 |
|---|---|
| 플레이어 이동 (Topdown3D) | 입력을 받아 캐릭터를 탑다운 3D 평면에서 이동시킨다 |
| 적 스폰 & 웨이브 | 시간·웨이브에 따라 적을 생성하고 수·강함을 스케일링한다 |
| 적 AI (추격) | 스폰된 적이 플레이어를 향해 이동한다 |
| 자동 공격 | 사거리 내 대상을 자동 탐색·공격한다 |
| 체력 & 피격 | 플레이어/적의 체력 관리, 피격·사망 처리 |
| 경험치 & 레벨업 | 처치 보상 수집, 레벨업 트리거, 능력 선택 제공 |

## Feature Priority

이 문서에서 가장 중요한 섹션이다.

### MVP — "한 판이 성립하는 최소 뱀서"

1. 플레이어 이동 (Topdown3D)
2. 적 스폰 (단순 연속 스폰, 웨이브 스케일링은 최소)
3. 적 추격 AI
4. 자동 공격 (단일 무기, 가장 가까운 적 타격)
5. 체력 & 피격 → 사망 시 게임오버
6. 경험치 & 레벨업 (레벨업 시 능력 1~3택)

### 나중 (Later)

- 다양한 무기·패시브와 무기 진화/합성
- 본격 웨이브 설계·보스전
- 저주(난이도↑ 보상↑) 시스템
- 메타 성장(영구 업그레이드), 상점/재화
- 다중 캐릭터, 스테이지
- 사운드·이펙트 폴리싱

## Development Order

1. **플레이어 이동** — 나머지 모든 것의 기반
2. **적 스폰 + 추격 AI** — 상대할 대상 확보
3. **체력 & 피격** — 플레이어/적이 죽을 수 있게
4. **자동 공격** — 적을 처치 가능하게 (피격 시스템 위에 얹힘)
5. **경험치 & 레벨업** — 처치에 보상·성장 부여, 루프 완성
6. **게임오버/결과 화면** — 한 판 마무리
