코드에 기능을 설명하는 주석을 달지 않는다.

**Why:** 사용자가 명시적으로 요청함. 잘 지어진 식별자 이름이 이미 코드의 의미를 전달하며, 주석은 불필요한 노이즈가 된다.

**How to apply:** C# 코드 작성 시 메서드·클래스·블록에 "이 기능은 ~를 한다"는 형태의 설명 주석을 추가하지 않는다. WHY가 비자명한 경우(숨겨진 제약, 특정 버그 우회)에만 한 줄 주석을 허용한다.
